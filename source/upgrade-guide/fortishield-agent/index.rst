Fortishield agent
===========

The following sections include instructions to upgrade the Fortishield agent to the latest available version. It is possible to upgrade the Fortishield agents either remotely from the Fortishield manager or locally. 

Upgrading the Fortishield agents remotely is possible by using the ``agent_upgrade`` tool and the Fortishield API. More information about the process can be found in the :doc:`Remote agent upgrade </user-manual/agents/remote-upgrading/upgrading-agent>` section.

To perform the upgrade locally, select your operating system and follow the instructions.
