Fortishield files backup
==================

In this section you can find instructions on how to create and restore a backup of your Fortishield installation.

To do this backup, you copy key files to a folder preserving file permissions, ownership, and path. Later, you can move this folder contents back to the corresponding location to restore your Fortishield data, certificates, and configurations. Backing up Fortishield files is useful in cases such as moving your Fortishield installation to another system.
